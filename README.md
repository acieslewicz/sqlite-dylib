# SQLite3 Dynamic Library Builder

A CMake-based build configuration for creating SQLite3 dynamic libraries.

## Build Dylib

`cmake -S . -B build`

`cmake --build build`

## Version info

This build uses the SQLite 3.6.10 amalgamation as a source.
